ARPES Python Tools

This set of libraries are being written to explore, analyze and visualize ARPES data. Currently, it can import data acquired by Scienta SES software format, but a data loader can be implemented for any other formats easily. It also allows importing and exporting Igor text format. There are functions to convert 2-dimensional spectra or 3-dimensional Fermi mapping data to k-space from the theta and/or phi space. There are basic matplotlib templates included in the documentation in order to quickly visualize your data. You can also slice volume data or extract line profiles from 2-dimensional image data using the libraries.

Integrate OneDrive cloud with your application

We can use Microsoft Graph API to integrate OneDrive or OneDrive for Business cloud storage with our programs to read write data from the cloud. If you have hundreds of gigabytes of data, it is not a bad idea to store them in the cloud, and access them only when you need. This is a walk-through on how to access, explore, read, write files to OneDrive using Python. In fact, Graph API can be used to access the whole of Microsoft services including Outlook emails and Excel online.
