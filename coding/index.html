Coding

Some of the coding projects I am working on.

ARPES Python Tools

This set of libraries are being written to explore, analyze and visualize ARPES data. Currently, it can import data acquired by Scienta SES software format, but a data loader can be implemented for any other formats easily. It also allows importing and exporting Igor text format. There are functions to convert 2-dimensional spectra or 3-dimensional Fermi mapping data to k-space from the theta and/or phi space. There are basic matplotlib templates included in the documentation in order to quickly visualize your data. You can also slice volume data or extract line profiles from 2-dimensional image data using the libraries.

SUV Python Tools

Python libraries to load, analyze and visualize SSLS SUV beamline data. There is also a web application to convert/export the SPEC/FOURC data formtat to CSV or plaintext formats.

ReTasks

This is an educational project to learn about React and Javascript in general. This application can helps you remember and manage your tasks. The user authentication and user data is stored in Google Firebase.

Linux/UNIX command cheatsheets

Never forget or confuse another Linux/UNIX command again! This is simple React webapp inspired by the cheat utility. The benefit of the webapp is that you don’t need to install in your device (provided you are connected to the internet when you need it), and it can be accessed from any device. The cheatsheets are based on the community version.

Singapore traffic camera

Monitor realtime traffic around you in Singapore. You can either select a location from the dropdown or enter GPS location, and it will show you the road image of nearest available traffic camera. If you want a specific location to be included in the dropdown menu, just drop me a message :)

Integrate OneDrive cloud with your application

We can use Microsoft Graph API to integrate OneDrive or OneDrive for Business cloud storage with our programs to read write data from the cloud. If you have hundreds of gigabytes of data, it is not a bad idea to store them in the cloud, and access them only when you need. This is a walk-through on how to access, explore, read, write files to OneDrive using Python. In fact, Graph API can be used to access the whole of Microsoft services including Outlook emails and Excel online.
